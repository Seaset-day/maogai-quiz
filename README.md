# 毛概题库 · 在线刷题

👉 **在线刷题（点开即用，手机/电脑都行）：** https://seaset-day.github.io/maogai-quiz/

> 觉得有用的话，点右上角 ⭐ **Star** 支持一下～

单文件网页题库，也可**下载 `index.html` 双击离线使用**，或发到手机浏览器打开。

## 功能
- **刷题练习**：单选 / 多选 / 判断自动判分，选项每次乱序（不背位置）；连续答对 2 次才"掌握"，答错进错题本并很快重现（Leitner 间隔重现）。
- **顺序浏览** + 答题卡，按题型筛选、错题筛选。
- **键盘作答**：1-4 / A-D 选项，Enter 提交/下一题。
- **📄 导出 Word**：点右上角按钮，可把整套题（含答案）导出成 `.doc`，带导出进度界面。
- 进度、错题、正确率保存在本机浏览器。

## 题量
单选 / 多选 / 判断 + 论述大题，合计 1130 题。

## 怎么用 / 怎么分享
- **最省事**：直接发上面的在线链接，对方点开就能刷。
- **离线**：`Code → Download ZIP` 或下载 `index.html`，双击打开。
